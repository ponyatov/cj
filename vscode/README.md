github: https://github.com/ponyatov/cj/tree/shadow/vscode

## Requirements

- VSCode >= 1.65.0
- OCaml
- Linux (Debian 12)

## Extension Settings

This extension contributes the following settings:

- `cj.Path`: The absolute path to toolchain root
